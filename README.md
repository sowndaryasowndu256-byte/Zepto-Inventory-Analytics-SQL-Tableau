# Zepto-Inventory-Analytics-SQL-Tableau
 Analyzed a real-world Zepto inventory dataset containing 3,000+ SKUs using SQL and Tableau. Performed data cleaning, exploratory analysis, and built interactive dashboards to uncover pricing trends, discount patterns, category performance, and inventory insights for business decision-making.


##  Project Overview

This project focuses on analyzing the **Zepto Inventory Dataset**, a real-world retail dataset containing over **3,000+ SKUs (Stock Keeping Units)** across multiple product categories. The analysis was performed using **SQL** for data exploration and **Tableau** for interactive data visualization.

The goal of this project is to uncover valuable business insights related to pricing, discounts, inventory availability, and category performance.



##  Objectives

- Analyze product pricing and discount trends.
- Identify inventory availability across categories.
- Understand category-wise product distribution.
- Discover high-value and low-stock products.
- Create an interactive dashboard for business decision-making.



##  Tools & Technologies

- **SQL**
- **Tableau**
- **CSV Dataset**
- **Data Cleaning**
- **Exploratory Data Analysis (EDA)**
